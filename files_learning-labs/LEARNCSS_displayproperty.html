CSS: the "display" property

  • Every element has a default display value (depending on type of element).
  • Default for most elements is block or inline.
  • Block element, most often called "block-level element"
  • Inline element, just called "inline element"

display: block

<div>

<div> is the standard block-level element. A block-level element starts on a new line and stretches out to the left and right as far as it can.

</div>

Common block-level elements:

  • <div>, <hr>
  • <p>, <pre>, <address>, <blockqoute>
  • <h1> ... <h6>
  • <ol>, <ul>
  • <form>, <table>
  • <header>, <section>, <article>, <main>, <footer>

display: inline-block

<span> is the standard inline element. An inline element can wrap some text inside a paragraph <span> like this </span> without disrupting the flow of that paragraph. The a element is the most common inline element, since you use them for links.

Common inline elements:

  • <b>, <big>, <i>, <small>
  • <abbr>, <acronym>, <cite>, <code>, <em>, <strong>
  • <a>, <br>, <img>, <span>
  • <button>, <input>, <label>, <textarea>

display: none

  • Common display value is display: none;
  • Some specialized elements such as script use this as their default.
  • Commonly used with JavaScript to hide and show elements without really deleting and recreating them.
  • Different from visibility, which hides the element, but still takes up space.
  • Setting display: none; render page as element does not exist.
